# 동탄 파라곤 3차 — 모바일 랜딩 페이지

경기도 화성시 동탄2지구 A58BL · 지하2층~지상20층 18개동 총 1,247세대 · 전용 82·108㎡
공공지원 민간임대주택(10년 임대 후 분양전환) 관심고객등록 랜딩 페이지.

## 구조
- `index.html` — 페이지 마크업/스타일/스크립트 (수정할 일 거의 없음)
- `config.js` — **여기만 수정**하면 문구·이미지·메뉴가 전부 바뀝니다
- `api/contact.js` — 관심고객등록 폼 메일 발송(Vercel 서버리스, nodemailer/Gmail)
- `assets/images/` — 섹션 이미지 (원본 사이트에서 저작권 허락 하에 사용)

## 대표번호
- 상담 대표번호: **1800-7081** (`config.js`의 `callPhone`)

```
npx serve -l 5060 .
```

## 배포 (GitHub + Vercel)
1. 이 폴더를 GitHub 저장소로 push
2. Vercel에서 Import → 환경변수 등록 (`.env.example` 참고)
   - `EMAIL_USER`, `EMAIL_PASS`(Gmail 앱 비밀번호), `RECEIVER_EMAILS`(콤마로 여러 명), `SITE_NAME`
3. 배포 후 실제 도메인을 `config.js`의 `seo.canonical`에 입력

평면/배치/조감 이미지는 인·허가 및 시공 과정에서 변경될 수 있습니다.
